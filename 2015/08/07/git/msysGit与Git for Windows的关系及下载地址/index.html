msysGit与Git for Windows的关系及下载地址

By Zhang Jianun

发表于 2015-08-07

文章目录

以前就没弄明白msysGit与Git for Window倒底是啥关系，现在也没整太明白。好在现在两个项目合并，弃用msysGit这个名字，改用Git for Windows。msysGit源码还在更新，但很久没更新可执行安装程序了，目前在1.9.5。Git for windows目前发布到了2.4.6 fifth candidate，正式版应该很快了。

Ps: 2015-08-31再来看已经是正式版了！Git for Windows 2.5.1

整理了以下链接：
两者的关系官方表述见：https://github.com/msysgit/msysgit/wiki/Relationship-to-Git-for-Windows
msysGit安装文件下载地址：http://git-scm.com/download/win
msysGit源码：https://github.com/msysgit/git/
Git for Window下载地址：http://git-for-windows.github.io/
Git for Window源码：https://github.com/git-for-windows/git
